## STEP 5 — HANDLE THE LOGO BACKGROUND ISSUE

The logo PNG has a solid black background. Test the result after Step 2A. One of these three approaches will look correct depending on the exact PNG:

**Option A (most likely correct) — mixBlendMode screen:**
```tsx
style={{ mixBlendMode: 'screen' }}
```
Black becomes transparent, white text remains white. Works perfectly on dark backgrounds.

**Option B — if screen doesn't work:**
```tsx
style={{ mixBlendMode: 'lighten' }}
```

**Option C — if the PNG actually has transparency (check file):**
```bash
file client/src/assets/svnteen-logo-white.png
# If it says "PNG" with alpha channel, no mixBlendMode needed at all
```
If the PNG has a transparent background already, remove the mixBlendMode style entirely and just use:
```tsx
<img src={logoImage} alt="Svnteen. TheResidency." className={`object-contain ${className}`} />
```

Try Option A first. If the logo appears with a black box around it, try Option B. If that also has issues, check for transparency with Option C.
